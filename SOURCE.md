This repository is a Vango/HappyHorse mirror of the GPT Image 2 prompt dataset used by the Vango site.

- Upstream repository recorded in the dataset: https://github.com/moosl/awsome-gpt-image-2-prompts

The upstream repository includes an MIT license. Prompt entries retain their original author and source URL fields when present.

Images in `images/prompts/` are mirrored copies of the main preview image URLs from the dataset so the Vango site is not dependent on third-party hotlinked image URLs.
